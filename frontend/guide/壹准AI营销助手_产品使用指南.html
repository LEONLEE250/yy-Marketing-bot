2. 安装与启动

下载安装包 → 双击安装 → 桌面快捷方式启动，三步完成。

1

下载安装包

从 GitHub Release 或联系官方人员获取安装文件

2

双击安装

一路点击「下一步」完成安装

3

启动应用

桌面快捷方式或开始菜单启动

⚠️ 重要：安装前请关闭杀毒软件

安装包和更新包可能会被 360、腾讯管家、火绒 等杀毒软件误判为病毒或风险软件。请务必在安装前关闭所有杀毒软件，安装完成后再开启。

后续每次更新版本时，同样需要先关闭杀毒软件再运行安装包，否则可能导致应用无法正常启动或功能异常。

3. 使用前准备

不同功能模块有不同依赖，提前准备好可以避免踩坑。

3.1 微信客户端

适用功能微信助手（群发中心 / 朋友圈）
要求电脑上需安装并登录微信 PC 客户端（4.1.8 版本）
说明多平台分发（抖音 / 视频号）不需要微信客户端

3.2 浏览器要求

适用功能多平台分发（抖音 / 视频号）
要求电脑上需安装 Chrome 或 Microsoft Edge 浏览器
说明Windows 10/11 系统自带 Edge，无需额外安装

3.3 AI 大模型配置

适用功能AI 创意中心（生文案 / 生图 / 生视频）
需要准备大模型的 API 地址、API Key、模型名称
说明首次使用需在「设置」页填写，详见第 8 节
注意：如果不使用 AI 创意中心功能，可以跳过此步骤。微信群发和朋友圈功能不需要大模型。

4. 多平台分发

当前支持抖音、视频号两大平台。一个视频、一份文案，同时发布到多个账号。后续将根据用户需要，新增其他平台。

4.1 账号管理

1

切换平台

顶部 Tab 选择「多平台分发」，点击「抖音」或「视频号」切换

2

添加并登录

点击「添加并登录」，输入账号名称 → 浏览器弹出扫码页 → 用手机扫码授权

3

刷新状态

点击「刷新状态」快速检查登录态，无需打开浏览器

4.2 发布视频

填写发布信息：

  • 标题（必填）：最多30字，视频号会自动过滤标点符号
  • 视频文件（必填）：MP4 / MOV / AVI / MKV
  • 描述（可选）：建议不超过1000字
  • 话题标签（可选）：逗号分隔，最多10个
  • 封面图片（可选）：建议 ≥720×1280
  • 发布时间：立即发布 或 定时发布
4

勾选目标账号 → 点击「发布」

应用会自动打开浏览器，逐个发布到勾选的账号。底部日志区可查看实时进度。

4.3 风险提示与注意事项

首次发布前建议在电脑前观察：首次使用多平台分发，建议全程在电脑前关注自动化流程。例如抖音首次发布时，页面可能会弹出滑块验证或短信验证码，这是抖音的安全机制，需要手动完成验证后才能正常发布。后续发布通常不会再出现。
发布过程中不要操作电脑：发布时会自动打开浏览器模拟人工操作。在此期间请勿移动鼠标或切换窗口，可能干扰自动化流程。
抖音定时发布限制：抖音仅支持 2 小时至 14 天后的定时发布。视频号无此限制。
Cookie（会话标识）有效期：登录后应用会保存 Cookie（一种用于记录登录状态的临时数据），下次发布无需重新扫码。视频号因微信自身机制，Cookie 有效周期可能较短（数小时到数天），如发现发布时跳到扫码页，在账号管理中重新登录即可。
推荐运行环境：运行多平台分发功能的电脑最好是 64位操作系统，同时预装 64位 Microsoft Edge 浏览器（Windows 10/11 自带的 Edge 通常就是 64 位）。32 位系统或 32 位浏览器可能存在兼容性问题。

5. 微信助手

顶部 Tab 选择「微信助手」，内部可切换「群发中心」和「朋友圈」两个板块。

5.1 群发中心

前置条件：电脑上需已登录微信 PC 客户端，且目标联系人在聊天列表中。

操作流程：

1上传图片 / 视频（可选）+ 填写消息文案
2点击「加载会话列表」获取微信聊天列表，勾选收件人（一次只能展示10~15个列表，取决于微信前端实时展示的收件人）
3选择「立即发送」或「定时发送」→ 点击发送按钮
注意：使用群发中心和朋友圈的定时功能时，请勿将两个功能的发布时间设置重叠。两者都采用模拟鼠标操作的方式，同时运行时可能出现冲突导致发送失败。

5.2 朋友圈

操作流程：

1输入朋友圈文字内容（最多 2000 字）
2上传图片（最多 9 张）或视频（最多 1 个）
3选择隐私设置（公开 / 私密）和发布模式（立即 / 定时）
4点击「发布朋友圈」
注意：朋友圈发布依赖屏幕坐标定位，不同电脑的 DPI 缩放设置可能影响定位精度。如果发布失败，请在桌面右键 →「显示设置」→「缩放与布局」中，将缩放比例调整为 100%，重启应用后再试。

6. AI 创意中心

三个子功能：「AI 生文案」「AI 生图」「AI 生视频」。对话式交互，输入需求即可生成。

💬

AI 生文案

输入营销需求，自动生成推广文案、广告语、产品描述

🎨

AI 生图

文字描述 → 图片，支持多种尺寸比例，可选参考图

🎬

AI 生视频

文字描述 → 短视频，支持文本 / 图片驱动视频生成

💬 AI 生文案 使用技巧与场景

  • 朋友圈营销文案：输入产品名称、卖点和目标人群，快速生成多条朋友圈文案，选中满意的一条，一键填入朋友圈发布
  • 群发话术：生成特定场景下的营销话术，如节日问候、促销活动、新品通知等，一键填入群发中心
  • 多风格切换：可要求文案风格为正式、幽默、温暖、紧迫等，满足不同客户群体

🎨 AI 生图 使用技巧与场景

  • 朋友圈配图：描述想要的画面（如"简约风格的产品展示图，白色背景"），生成后可直接下载用于朋友圈发布
  • 营销海报：生成节日促销海报、活动宣传图，支持指定横版/竖版、比例尺寸
  • 参考图上传：可以上传参考图片，让 AI 基于参考图风格生成新内容

🎬 AI 生视频（敬请期待）

当前状态：AI 生视频模块目前还不稳定，正在优化升级中。如遇到生成失败、生成质量不佳等情况属于正常现象。我们正在全力优化，敬请期待后续版本。
需要先在「设置」页配置大模型：生文案需要语言大模型，生图生视频需要对应的图像 / 视频生成模型。详见第 8 节。
重要：每次重启应用后，需前往「设置」→ 点击对应模型的「测试连接」按钮，确认显示「连接正常」后再保存设置，否则 AI 功能可能无法正常使用。

7. 资源库

统一管理文字、图片、视频素材。支持分类、搜索、一键填入群发和朋友圈。

素材管理

  • 上传文案：在资源库面板中点击「新建文案」，输入标题和内容，保存后即可在群发和朋友圈中直接选用
  • 上传图片/视频：点击「上传素材」选择本地文件，支持常见格式（图片：JPG/PNG/WebP，视频：MP4/MOV/AVI）
  • 标签分类：可以为每个素材添加标签（如"春节促销"、"新品推广"、"节日问候"），通过标签快速筛选查找素材
  • 配置存储目录：在「设置」→「资源库」中可修改素材的本地存储路径，默认在 %APPDATA%\yizhun-wechat-bot-preview\resources

在微信助手中的快捷使用

  • 在群发中心或朋友圈页面，点击右上方的「搜索」图标，即可搜索资源库中的文案、标签，找到后直接选用
  • 编辑、删除、修改标签需要在资源库 Tab 中进行：切换到「资源库」面板，点击素材旁的编辑按钮进行修改
  • 搜索到的文案可一键填入内容输入框，图片/视频可添加到待发送素材列表

9. 常见问题

Q: 抖音发布时弹出验证码怎么办？

A: 这是抖音的安全验证，首次登录时偶尔出现。手动完成一次验证即可，后续发布不会再弹。如果频繁出现，建议在手机抖音 App 中完成一次完整登录后再试。

Q: 视频号显示「已登录」但实际发布时跳到扫码页？

A: Cookie 可能已过期或不完整。在账号管理中删除该账号，重新扫码登录一次即可。

Q: 刷新状态时原本删除的账号又出现了？

A: 旧版本的已知问题，v4.0 已修复。如果仍有残留，手动删除 %APPDATA%\yizhun-wechat-bot-preview\platform-cookies\shipinhao\ 目录下的 JSON 文件。

Q: AI 生成的内容没有反应？

A: 检查设置页的大模型配置是否正确（API 地址、Key、模型名）。确认 API Key 未过期且有足够余额。

Q: 群发发送失败？

A: 确认微信 PC 客户端已登录且未最小化到托盘。检查目标联系人是否在聊天列表中。尝试减少发送间隔时间。

Q: 视频号发布卡在上传页面不动？

A: 可能视频文件较大导致上传时间较长，耐心等待 1-2 分钟。超过 3 分钟仍未继续，请关闭弹出浏览器后重试。
