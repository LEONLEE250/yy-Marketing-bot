2. 安装与启动

下载安装包 → 双击安装 → 桌面快捷方式启动，三步完成。

  1. 下载安装包 — 从 GitHub Release 或官网获取 .exe 安装文件
  2. 双击安装 — 一路点击「下一步」完成安装
  3. 启动应用 — 桌面快捷方式或开始菜单启动

3. 使用前准备

不同功能模块有不同依赖，提前准备好可以避免踩坑。

3.1 微信客户端

适用功能微信助手（群发中心 / 朋友圈）
要求电脑上需安装并登录微信 PC 客户端（3.9.x 以上版本）
说明多平台分发（抖音 / 视频号）不需要微信客户端

3.2 浏览器要求

适用功能多平台分发（抖音 / 视频号）
要求电脑上需安装 Chrome 或 Microsoft Edge 浏览器
说明Windows 10/11 系统自带 Edge，无需额外安装

3.3 AI 大模型配置

适用功能AI 创意中心（生文案 / 生图 / 生视频）
要求需要准备大模型的 API 地址、API Key、模型名称
说明首次使用需在「设置」页填写，详见第 8 节
注意： 如果不使用 AI 创意中心功能，可以跳过此步骤。微信群发和朋友圈功能不需要大模型。

4. 多平台分发

支持抖音、视频号两大平台。一个视频、一份文案，同时发布到多个账号。

4.1 账号管理

  1. 切换平台 — 顶部 Tab 选择「多平台分发」，点击「抖音」或「视频号」切换
  2. 添加并登录 — 点击「添加并登录」，输入账号名称 → 浏览器弹出扫码页 → 用手机扫码授权
  3. 刷新状态 — 点击「刷新状态」快速检查登录态，无需打开浏览器

4.2 发布视频

填写发布信息：

  • 标题（必填）：最多30字，视频号会自动过滤标点符号
  • 视频文件（必填）：MP4 / MOV / AVI / MKV
  • 描述（可选）：建议不超过1000字
  • 话题标签（可选）：逗号分隔，最多10个
  • 封面图片（可选）：建议 ≥720×1280
  • 发布时间：立即发布 或 定时发布

勾选目标账号 → 点击「发布」。应用会自动打开浏览器，逐个发布到勾选的账号。底部日志区可查看实时进度。

4.3 风险提示与注意事项

抖音首次登录可能需要验证码： 部分抖音账号在首次扫码后，页面会弹出滑块验证或短信验证码。这是抖音的安全机制，需要手动完成一次。后续发布不会再出现。
发布过程中不要操作电脑： 发布时会自动打开浏览器模拟人工操作。在此期间请勿移动鼠标或切换窗口，可能干扰自动化流程。
抖音定时发布限制： 抖音仅支持 2 小时至 14 天后的定时发布。视频号无此限制。
Cookie 有效期： 登录一次后 Cookie 会保存在本地，下次发布无需重新扫码。如果长时间未使用导致 Cookie 过期，只需重新扫码登录即可。

5. 微信助手

顶部 Tab 选择「微信助手」，内部可切换「群发中心」和「朋友圈」两个板块。

5.1 群发中心

前置条件： 电脑上需已登录微信 PC 客户端，且目标联系人在聊天列表中。
  1. 上传图片 / 视频（可选）+ 填写消息文案
  2. 点击「加载会话列表」获取微信聊天列表，勾选收件人
  3. 选择「立即发送」或「定时发送」→ 点击发送按钮

5.2 朋友圈

  1. 输入朋友圈文字内容（最多 2000 字）
  2. 上传图片（最多 9 张）或视频（最多 1 个）
  3. 选择隐私设置（公开 / 私密）和发布模式（立即 / 定时）
  4. 点击「发布朋友圈」
注意： 朋友圈发布依赖屏幕坐标定位，不同电脑的 DPI 缩放设置可能影响定位精度。如果发布失败，请在设置中调整相关参数。

6. AI 创意中心

三个子功能：「AI 生文案」「AI 生图」「AI 生视频」。对话式交互，输入需求即可生成。

💬
AI 生文案
输入营销需求，自动生成推广文案、广告语、产品描述
🎨
AI 生图
文字描述 → 图片，支持多种尺寸比例，可选参考图
🎬
AI 生视频
文字描述 → 短视频，支持文本 / 图片驱动视频生成
需要先在「设置」页配置大模型： 生文案需要语言大模型，生图生视频需要对应的图像 / 视频生成模型。详见第 8 节。

7. 资源库

统一管理文字、图片、视频素材。支持分类、搜索、标签管理、一键填入群发和朋友圈。

  • 新建资源 — 手动添加文案、图片、视频，可设置标题和标签
  • 批量导入 — 一键导入文件夹中的图片和视频文件，自动以文件名命名
  • 标签管理 — 创建预设标签，方便分类筛选
  • 一键使用 — 在群发中心或朋友圈编辑时，点击「📦 资源库」按钮选取素材

9. 常见问题

抖音发布时弹出验证码怎么办？
这是抖音的安全验证，首次登录时偶尔出现。手动完成一次验证即可，后续发布不会再弹。如果频繁出现，建议在手机抖音 App 中完成一次完整登录后再试。
视频号显示「已登录」但实际发布时跳到扫码页？
Cookie 可能已过期或不完整。在账号管理中删除该账号，重新扫码登录一次即可。
刷新状态时原本删除的账号又出现了？
旧版本的已知问题，v4.0 已修复。如果仍有残留，手动删除 %APPDATA%\yizhun-wechat-bot-preview\platform-cookies\shipinhao\ 目录下的 JSON 文件。
AI 生成的内容没有反应？
检查设置页的大模型配置是否正确（API 地址、Key、模型名）。确认 API Key 未过期且有足够余额。
群发发送失败？
确认微信 PC 客户端已登录且未最小化到托盘。检查目标联系人是否在聊天列表中。可以尝试增大发送间隔时间。
视频号发布卡在上传页面不动？
可能视频文件较大导致上传时间较长，耐心等待 1-2 分钟。超过 3 分钟仍未继续，请关闭弹出浏览器后重试。
批量导入为什么选不了视频文件？
v4.0.6 已修复此问题。如果仍有问题，请确保文件对话框的筛选器选择了「图片和视频」类型。
